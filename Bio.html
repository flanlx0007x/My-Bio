About Me

ผมชื่อแฟลงค์ อายุ 15 ปี เพศชาย ส่วนสูงอยู่ระหว่าง 160 ถึงเกือบ 170 ซม. ชอบเขียนโค้ดและคณิต เพราะมันเท่สุดๆ เมื่อทำได้

ผมเขียนโค้ดได้ถึง 4 ภาษา ได้แก่ Python, HTML (ถึงจะไม่นับว่าเป็นภาษาโปรแกรมเมอร์ก็เถอะ), CSS (นี้ก็อาจไม่ต่างกันมาก) และ JavaScript

ผมคือโปรแกรมเมอร์ที่หลงใหลในการเขียนโค้ดและสร้างสรรค์สิ่งใหม่ๆ ด้วยพลังของเทคโนโลยี ผมเริ่มต้นเส้นทางนี้ด้วยความสนใจใน Python ซึ่งตอนนี้กลายเป็นสกิลที่ผมเชี่ยวชาญมากที่สุด ด้วยระดับความสามารถถึง 70% ผมใช้ Python ในการพัฒนาโปรเจกต์ตั้งแต่การวิเคราะห์ข้อมูลไปจนถึงการสร้างโมเดลการเรียนรู้ของเครื่องที่ซับซ้อน

นอกจากนี้ ผมยังเริ่มต้นเรียนรู้และฝึกฝนทักษะด้านการพัฒนาเว็บ ด้วย HTML, CSS และ JavaScript ซึ่งแต่ละอย่างผมให้คะแนนตัวเองที่ 10% เพราะผมเชื่อว่าการเรียนรู้ไม่มีที่สิ้นสุด และนี่เป็นเพียงจุดเริ่มต้นของการเดินทางเพื่อเป็นนักพัฒนาแบบเต็มสแตก
